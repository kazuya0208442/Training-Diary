# 📱 Training-Diary(App) 

オリンピック強化選手になると、**運動日誌の提出**が義務付けられます。ただ、いくつか不便な点があります。
- 記入欄が狭すぎる
- 字が読み辛い
- 過去のデータと比較が困難
- 紛失リスク


この問題を解決するべく、**運動日誌アプリ**を製作しました。:point_right:  &nbsp;[http://kazuya-portfolio.com](http://kazuya-portfolio.com/)  　


アプリ内の記入例は**当時の私の運動日誌**をそのまま使ってます(^^♪
